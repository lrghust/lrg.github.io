Implement the method of Seq-NMS

Created at 2017-02-19 Updated at 2018-02-25 Category SeqNMS / Code Tag SeqNMS / Video Object Detection / Code

I implement the method Seq-NMS mentioned in paper: SEQ-NMS FOR VIDEO OBJECT DETECTION based on the Faster-RCNN. You can go to my github to see it.

Without a good computer to train a network, I use the demo program and pre-trained network given by Faster-RCNN to do the detection in each frame in a video.

If you want to use it, follow the demo instruction given by Faster-RCNN, and replace the demo.py file by the video.py given here.

PS: It will be very very slow to detect a video…
